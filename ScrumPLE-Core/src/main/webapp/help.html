**Note: Returned collections are UUID->object maps** # Project ``` { name: (string), description: (string), owner: (User), isPrivate: (boolean) } ``` ### GET `/rest/projects` retrieves all **public** projects `/rest/projects/{ID}` retrieves the project referenced by `ID` ### POST Project object to `/rest/projects` creates a new project **if a project of the same name does not yet exist** and returns its UUID ### PUT Project object to `/rest/projects/{ID}` updates the project referenced by `ID` ### DELETE - Requires **project owner** authorization `/rest/projects/{ID}` deletes the project referenced by `ID` --- # User ``` { displayName: (string), role: (string) } ``` ### GET `rest/projects/{ID}/users` retrieves all users local to the project referenced by `ID` --- # Meeting ``` { type: (string), start: (long), end: (long) } ``` ### GET `rest/projects/{ID}/meetings` retrieves all meetings local to the project referenced by `ID`
